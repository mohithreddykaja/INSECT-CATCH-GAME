# Insect Catch Game

A simple browser game built with HTML, CSS, and JavaScript. Catch the insects by clicking them before time runs out.

## How to run

1. Open `index.html` in your browser, or run a local HTTP server:
   ```bash
   python -m http.server 8000
   ```
2. Visit `http://localhost:8000`

## Files

- `index.html` - game structure and layout
- `style.css` - game styling
- `script.js` - game logic

## Notes

- This project is designed for learning and practice.
- Use a modern browser for the best experience.
